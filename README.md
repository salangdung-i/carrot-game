## 바닐라JS를 이용해 만드는 carrot game🎮

시간내에 모든 당근을 클릭하면 다음단계로 넘어가는 미니게임입니다. <br>
총 3단계로 이루어져 있으며, 실행중인 단계에서 실패할 경우 현재의 단계를 다시 실행 할 수 있습니다.

</figure>

## 프로젝트 소개 
 ► 버튼을 누르면 게임이 시작됩니다.  <br>
 🥕을 클릭하면 🥕이 삭제됩니다.  <br>
 시간 내에 모든 🥕을 제거하면 🏆 우승합니다.  <br>
 단,🐛누르면 게임에서 지게됩니다.  

### 기술스택 
- HTML, CSS, JAVASCRIPT
